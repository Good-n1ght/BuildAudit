# BuildAudit 硬件对比助手

BuildAudit 是一个装机配置对比与评分工具，目标是帮助用户更直观地判断一套电脑配置是否值得买、是否搭配合理、是否存在兼容风险。

它主要面向两类人：

- 装机小白：快速看懂 CPU、显卡、主板、内存、硬盘、电源、散热等核心配件是否匹配。
- 硬件玩家：通过整机跑分、等级颜色和结果图，获得更有展示感的配置评分结果。

## 核心功能

- 多硬件配件搜索与选择
- 整机评分与分项评分
- 配件性能、性价比、搭配合理性分析
- 兼容性风险提示
- 装机建议与问题提醒
- 等级颜色视觉系统
- 结果面板与分享图方向的展示设计

## 评分方向

当前评分会综合考虑：

- 单个硬件性能
- 不同硬件之间的搭配关系
- 价格与性价比
- 平台定位
- 新手装机风险
- 整机配置均衡度

评分系统仍在持续调整中，后续会继续优化权重、硬件数据库和兼容性规则。

## 项目状态

这是一个早期原型版本，目前已经可以在浏览器中运行，但硬件数据库、评分权重、兼容性判断和结果图展示仍在持续完善。

后续重点方向：

- 扩充真实硬件型号数据库
- 增强主板、显卡、散热、电源之间的兼容判断
- 优化整机评分权重
- 改进适合分享的装机跑分结果图
- 增加更清晰的新手提示

## 本地运行

直接用浏览器打开 `index.html` 即可运行。

## 主要文件

- `index.html`：页面结构与样式
- `app.js`：应用逻辑、搜索、评分和结果渲染
- `data.js`：硬件数据库
