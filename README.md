# 硬件对比助手 BuildAudit

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> 装机不踩坑：本地硬件数据库 + 智能打分引擎 + 配置跑分对比，一站式看清每一分钱花得值不值。

## 这是什么

BuildAudit 是一个**本地运行的硬件选配对比工具**。帮你在攒机时快速查参数、比性能、算分数，避免被营销话术带偏。

**适合谁用**：DIY 装机玩家、硬件爱好者、帮朋友写配置单的人。

## 能做什么

- **硬件数据库** — 收录 CPU 138 款、GPU 3116 款、主板 3109 款等共 9923 条硬件信息
- **搜索筛选** — 按关键词、品牌、接口类型快速定位硬件
- **多款对比** — 选中多款硬件，一键生成规格对比表
- **跑分分析** — 自动计算 CPU/GPU 单件分，用颜色分级直观显示档次
- **装机综合评分** — 选完整套配置后，生成总分和配件评级（曜黑暗金 → 白色普通七级）
- **洋垃圾/兼容性警告** — 自动标记 E5 洋垃圾、接口不匹配等坑点

## 怎么打开

**不需要安装任何软件，有浏览器就能用。**

1. 下载本项目到本地
2. 双击 `index.html`，用浏览器打开
3. 就开始用了

> 如果想用搜索功能（`engine/search.py`），需要 Python 3.8+ 环境。

## 目录结构

```
硬件对比_Codex工作区/
├── index.html          # 主界面（双击打开即用）
├── app.js              # 前端逻辑：搜索/筛选/对比/评分渲染
├── data.js             # 硬件数据库
├── engine/             # Python 引擎：搜索/打分/兼容性检查
├── api/                # API 层
├── updater/            # 数据维护：校验/索引/备份
├── docs/               # 设计文档
└── README.md
```

## 开发相关

```bash
# 构建搜索索引
python -m updater.maintain --rebuild-index

# 校验数据库
python -m updater.maintain --validate

# 测试搜索
python engine/search.py
```
