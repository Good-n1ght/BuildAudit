# 品牌梯队标准

> 所有 8 大件的品牌分级依据，供后续新增品牌时参考。

## 梯队定义

| 梯队 | 标准 |
|---|---|
| **T1** | 行业标杆 / 原厂 / 品质口碑顶级 |
| **T2** | 主流可靠品牌，正规质保，市场占有率高 |
| **T3** | 性价比/新兴/小众，有正规质保，非杂牌 |
| **不入流** | 无质保保障的杂牌 → is_e_waste=true |

## 各模块梯队

### CPU
不分品牌梯队。Intel / AMD 两家，按型号定 tier_score。

### GPU
不分品牌梯队，按芯片型号定 tier_score。

### 主板
按芯片组 + 品牌综合定 tier_score，品牌分档参考：
- T1：华硕 ROG / 微星 MEG / 技嘉 AORUS
- T2：华硕 TUF / 微星 MPG / 技嘉 GAMING / 华擎
- T3：铭瑄 / 七彩虹 / 昂达 / 精粤

### 内存
- T1：金士顿 / 芝奇 / 海盗船 / 宏碁掠夺者 / 威刚
- T2：光威 / 阿斯加特 / 雷克沙 / 宇瞻 / 十铨 / 科赋
- T3：博帝 / 金邦 / 创见 / 技嘉 AORUS / 华硕 ROG 等

### 散热器

### 机箱
